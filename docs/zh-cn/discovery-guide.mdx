xcsh浏览器自动化工具如何通过爬取F5 XC控制台来填充目录。

## 概述

目录条目使用**浏览器优先发现**方法创建：

1. xcsh连接到用户的Chrome浏览器（通过端口9222的CDP）
2. xcsh导航控制台并提取结构信息
3. xcsh输出与目录模式匹配的YAML文件
4. 人工审查、完善和提交条目

## 发现后验证

每次发现运行后：

1. 运行 `npm run validate` 检查模式合规性
2. 检查生成的条目是否准确
3. 调整置信度级别
4. 通过标准PR工作流提交和推送
