## 目的

此存储库是F5 Distributed Cloud（F5 XC）管理控制台UI的**模式驱动、机器可读清单**。它将API资源映射到其控制台位置、导航路径、表单结构和逐步浏览器自动化工作流。

目录由xcsh的浏览器自动化运行时使用，以支持：

- **销售演示** — 控制台CRUD操作的AI辅助演练
- **培训视频** — YouTube和演示文稿的可重现浏览器录制
- **UAT/QA自动化** — 自动化控制台交互测试
- **AI辅助导航** — AI助手可以查找在哪里创建资源并将浏览器导航到那里

## 集成点

| 系统 | 集成 |
|--------|-------------|
| api-specs-enriched | 资源 `api.kind` 值匹配；通过字符串键交叉引用 |
| xcsh browser.ts | 运行时消费者 — 读取YAML，通过CDP执行 |
| docs-control | 治理 — 托管文件、CI工作流、分支保护 |
| docs-theme | 从目录生成文档 |
