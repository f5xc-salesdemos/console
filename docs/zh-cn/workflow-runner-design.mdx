读取此目录中的工作流YAML并通过Chrome DevTools协议执行步骤的xcsh组件设计。

## 操作映射

| 工作流操作 | 浏览器工具操作 | 说明 |
|---|---|---|
| `navigate` | `goto` + `wait_for_selector` | 在相对URL前添加baseUrl |
| `click` | `click` | 直接选择器映射 |
| `fill` | `fill` | 解析值中的{param} |
| `assert` | `get_text` + 字符串比较 | 不匹配时立即失败 |

## 错误处理

- 快速失败：第一个失败的步骤停止执行
- 失败截图（尽力而为）
- 无重试逻辑
