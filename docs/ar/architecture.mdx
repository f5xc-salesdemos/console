## الغرض

هذا المستودع هو **مخزون قابل للقراءة آليًا ومدفوع بالمخطط** لواجهة مستخدم وحدة تحكم الإدارة في F5 Distributed Cloud (F5 XC). يربط موارد API بمواقعها في وحدة التحكم ومسارات التنقل وهياكل النماذج وسير عمل أتمتة المتصفح خطوة بخطوة.

يُستهلك الكتالوج من قِبل وقت تشغيل أتمتة المتصفح في xcsh لتمكين:

- **عروض المبيعات** — جولات إرشادية بمساعدة الذكاء الاصطناعي لعمليات CRUD في وحدة التحكم
- **مقاطع التدريب** — تسجيلات متصفح قابلة للإعادة لـ YouTube والعروض التقديمية
- **أتمتة UAT/QA** — اختبارات تفاعل وحدة التحكم المؤتمتة
- **التنقل بمساعدة الذكاء الاصطناعي** — يمكن للمساعد الذكي البحث عن مكان إنشاء مورد وتوجيه المتصفح إليه

## نقاط التكامل

| النظام | التكامل |
|--------|-------------|
| api-specs-enriched | قيم `api.kind` للموارد متطابقة؛ مرجعية تبادلية بمفتاح سلسلة |
| xcsh browser.ts | مستهلك وقت التشغيل — يقرأ YAML، ينفذ عبر CDP |
| docs-control | الحوكمة — الملفات المُدارة، سير عمل CI، حماية الفروع |
| docs-theme | توليد الوثائق من الكتالوج |
