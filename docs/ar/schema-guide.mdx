كيفية تأليف وصيانة إدخالات الكتالوج.

## مستويات الثقة

| المستوى | المعنى | متى تُستخدم |
|-------|---------|-------------|
| `draft` | غير مُتحقق | التأليف الأولي، الإدخالات المكتشفة تلقائيًا |
| `validated` | مؤكد | تم التحقق منه مقابل وحدة التحكم الحية في الإصدار المحدد |
| `stale` | قديم | تغيرت وحدة التحكم منذ آخر تحقق |

## التحقق

شغّل `npm run validate` للتحقق من جميع إدخالات الكتالوج مقابل مخططاتها.
