كيف تملأ أداة أتمتة المتصفح xcsh الكتالوج عن طريق زحف وحدة تحكم F5 XC.

## نظرة عامة

يتم إنشاء إدخالات الكتالوج باستخدام نهج **اكتشاف المتصفح أولاً**:

1. يتصل xcsh بمتصفح Chrome الخاص بالمستخدم (عبر CDP على المنفذ 9222)
2. يتنقل xcsh في وحدة التحكم ويستخرج المعلومات الهيكلية
3. يُنتج xcsh ملفات YAML تطابق مخططات الكتالوج
4. يراجع إنسان الإدخالات ويصقلها ويُثبتها

## التحقق بعد الاكتشاف

بعد كل تشغيل اكتشاف:

1. شغّل `npm run validate` للتحقق من الامتثال للمخطط
2. راجع الإدخالات المُنشأة للتحقق من دقتها
3. اضبط مستويات الثقة
4. ثبّت وادفع عبر سير عمل PR القياسي
