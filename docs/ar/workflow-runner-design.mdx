تصميم مكوّن xcsh الذي يقرأ YAML لسير العمل من هذا الكتالوج وينفذ الخطوات عبر بروتوكول Chrome DevTools.

## تعيين الإجراءات

| إجراء سير العمل | إجراء Browser Tool | ملاحظات |
|---|---|---|
| `navigate` | `goto` + `wait_for_selector` | إلحاق baseUrl بعنوان URL النسبي |
| `click` | `click` | تعيين مباشر للمحدد |
| `fill` | `fill` | حل {param} في القيمة |
| `assert` | `get_text` + مقارنة سلسلة | فشل فوري عند عدم التطابق |

## معالجة الأخطاء

- الفشل الفوري: أول خطوة فاشلة تُوقف التنفيذ
- لقطة شاشة للفشل (أفضل جهد)
- بدون منطق إعادة المحاولة
