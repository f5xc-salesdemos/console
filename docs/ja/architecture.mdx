## 目的

このリポジトリは、F5 Distributed Cloud（F5 XC）管理コンソールUIの**スキーマ駆動、機械可読インベントリ**です。APIリソースをコンソールの場所、ナビゲーションパス、フォーム構造、ステップバイステップのブラウザ自動化ワークフローにマッピングします。

カタログはxcshのブラウザ自動化ランタイムによって消費され、以下を実現します：

- **セールスデモ** — コンソールCRUD操作のAI支援ウォークスルー
- **トレーニングビデオ** — YouTubeとプレゼンテーション用の再現可能なブラウザ録画
- **UAT/QA自動化** — コンソールインタラクションの自動化テスト
- **AI支援ナビゲーション** — AIアシスタントがリソースを作成する場所を検索し、ブラウザをそこに誘導できる

## 統合ポイント

| システム | 統合 |
|--------|-------------|
| api-specs-enriched | リソースの `api.kind` 値が一致; 文字列キーによる相互参照 |
| xcsh browser.ts | ランタイムコンシューマー — YAMLを読み込み、CDP経由で実行 |
| docs-control | ガバナンス — 管理ファイル、CIワークフロー、ブランチ保護 |
| docs-theme | カタログからのドキュメント生成 |
