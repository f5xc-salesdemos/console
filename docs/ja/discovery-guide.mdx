xcshブラウザ自動化ツールがF5 XCコンソールをクロールしてカタログを作成する方法。

## 概要

カタログエントリは**ブラウザファーストディスカバリー**アプローチを使用して作成されます：

1. xcshはユーザーのChromeブラウザに接続します（ポート9222のCDP経由）
2. xcshはコンソールをナビゲートし、構造情報を抽出します
3. xcshはカタログスキーマに一致するYAMLファイルを出力します
4. 人間がエントリをレビュー、洗練、コミットします

## ディスカバリー後の検証

各ディスカバリー実行後：

1. `npm run validate` を実行してスキーマコンプライアンスを確認
2. 生成されたエントリの精度をレビュー
3. 信頼度レベルを調整
4. 標準PRワークフロー経由でコミットおよびプッシュ
