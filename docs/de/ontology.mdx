Definitionen der im Konsolenkatalog verwendeten Begriffe.

## Kernbegriffe

### Route
Ein Konsolen-URL-Pfad oder -Muster, das einen bestimmten Bildschirm identifiziert.

### Ressource
Ein API-definierter Objekttyp mit bekanntem Schema, CRUD-Endpunkten und einer oder mehreren Konsolenstandorten.

### Workflow
Eine geordnete Abfolge von Automatisierungsschritten, die genau eine Operation an einer Ressource ausführt.

### Schritt
Eine einzelne Automatisierungsaktion in einem Workflow. Aktionen umfassen: navigate, click, fill, select, assert, screenshot.

## Metadatenbegriffe

### Konfidenz
Validierungsstatus eines Katalogeintrags:
- **draft** — erstellt oder auto-entdeckt, aber noch nicht validiert
- **validated** — für eine bestimmte Konsolenversion als korrekt bestätigt
- **stale** — zuvor validiert, aber die Konsole hat sich seitdem geändert
