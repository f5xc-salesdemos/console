เครื่องมือทำงานอัตโนมัติของเบราว์เซอร์ xcsh เติมข้อมูลแคตตาล็อกโดยการรวบรวมคอนโซล F5 XC อย่างไร

## ภาพรวม

รายการแคตตาล็อกถูกสร้างขึ้นโดยใช้วิธีการ **ค้นหาแบบเบราว์เซอร์เป็นหลัก**:

1. xcsh เชื่อมต่อกับเบราว์เซอร์ Chrome ของผู้ใช้ (ผ่าน CDP บนพอร์ต 9222)
2. xcsh นำทางในคอนโซลและดึงข้อมูลโครงสร้าง
3. xcsh ส่งออกไฟล์ YAML ที่ตรงกับสคีมาแคตตาล็อก
4. มนุษย์ตรวจสอบ ปรับปรุง และ commit รายการ

## การตรวจสอบหลังการค้นหา

หลังจากรันการค้นหาแต่ละครั้ง:

1. เรียกใช้ `npm run validate` เพื่อตรวจสอบการปฏิบัติตามสคีมา
2. ตรวจสอบรายการที่สร้างขึ้นเพื่อความถูกต้อง
3. ปรับระดับความเชื่อมั่น
4. Commit และ push ผ่านเวิร์กโฟลว์ PR มาตรฐาน
