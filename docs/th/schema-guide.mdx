วิธีสร้างและดูแลรักษารายการแคตตาล็อก

## ระดับความเชื่อมั่น

| ระดับ | ความหมาย | เมื่อใดที่ใช้ |
|-------|---------|-------------|
| `draft` | ยังไม่ได้ตรวจสอบ | การสร้างเริ่มต้น รายการที่ค้นพบอัตโนมัติ |
| `validated` | ยืนยันแล้ว | ตรวจสอบกับคอนโซลที่ทำงานอยู่ในเวอร์ชันที่ระบุ |
| `stale` | ล้าสมัย | คอนโซลเปลี่ยนแปลงหลังจากการตรวจสอบครั้งล่าสุด |

## การตรวจสอบความถูกต้อง

เรียกใช้ `npm run validate` เพื่อตรวจสอบรายการแคตตาล็อกทั้งหมดกับสคีมาของพวกเขา
