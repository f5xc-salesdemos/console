คำจำกัดความของคำที่ใช้ในแคตตาล็อกคอนโซล

## คำหลัก

### เส้นทาง
เส้นทาง URL ของคอนโซลหรือรูปแบบที่ระบุหน้าจอที่แตกต่างกัน

### ทรัพยากร
ประเภทออบเจ็กต์ที่กำหนดโดย API ที่มีสคีมาที่รู้จัก จุดสิ้นสุด CRUD และตำแหน่งหนึ่งตำแหน่งขึ้นไปในคอนโซล

### เวิร์กโฟลว์
ลำดับขั้นตอนการทำงานอัตโนมัติที่เรียงลำดับซึ่งดำเนินการหนึ่งอย่างบนทรัพยากรหนึ่งรายการ

## คำเมตาดาต้า

### ความเชื่อมั่น
สถานะการตรวจสอบของรายการแคตตาล็อก:
- **draft** — สร้างหรือค้นพบอัตโนมัติแต่ยังไม่ได้ตรวจสอบ
- **validated** — ยืนยันว่าถูกต้องสำหรับเวอร์ชันคอนโซลที่ระบุ
- **stale** — ตรวจสอบก่อนหน้านี้แต่คอนโซลเปลี่ยนแปลงแล้ว
