## จุดประสงค์

ที่เก็บนี้เป็น **คลังข้อมูลที่อ่านได้ด้วยเครื่องและขับเคลื่อนด้วยสคีมา** ของ UI คอนโซลผู้ดูแลระบบ F5 Distributed Cloud (F5 XC) โดยจับคู่ทรัพยากร API กับตำแหน่งในคอนโซล เส้นทางการนำทาง โครงสร้างฟอร์ม และเวิร์กโฟลว์การทำงานอัตโนมัติของเบราว์เซอร์แบบทีละขั้นตอน

แคตตาล็อกถูกใช้โดยรันไทม์การทำงานอัตโนมัติของเบราว์เซอร์ xcsh เพื่อสนับสนุน:

- **การสาธิตการขาย** — คำแนะนำแบบ AI สำหรับการดำเนินการ CRUD ในคอนโซล
- **วิดีโอฝึกอบรม** — การบันทึกเบราว์เซอร์ที่ทำซ้ำได้สำหรับ YouTube และการนำเสนอ
- **การทำงานอัตโนมัติ UAT/QA** — การทดสอบอัตโนมัติสำหรับการโต้ตอบกับคอนโซล
- **การนำทางด้วย AI** — ผู้ช่วย AI สามารถค้นหาตำแหน่งที่จะสร้างทรัพยากรและนำทางเบราว์เซอร์ไปที่นั่น

## จุดผสานรวม

| ระบบ | การผสานรวม |
|--------|-------------|
| api-specs-enriched | ค่า `api.kind` ของทรัพยากรตรงกัน; อ้างอิงไขว้ด้วยคีย์สตริง |
| xcsh browser.ts | ผู้บริโภครันไทม์ — อ่าน YAML ดำเนินการผ่าน CDP |
| docs-control | การกำกับดูแล — ไฟล์ที่จัดการ เวิร์กโฟลว์ CI การป้องกันสาขา |
| docs-theme | การสร้างเอกสารจากแคตตาล็อก |
