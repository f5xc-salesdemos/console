Definições de termos usados no catálogo do console.

## Termos principais

### Rota
Um caminho de URL do console ou padrão que identifica uma tela distinta.

### Recurso
Um tipo de objeto definido por API com esquema conhecido, endpoints CRUD e um ou mais locais no console.

### Fluxo de trabalho
Uma sequência ordenada de etapas de automação que realiza exatamente uma operação em um recurso.

### Etapa
Uma única ação de automação em um fluxo de trabalho. As ações incluem: navigate, click, fill, select, assert, screenshot.

## Termos de metadados

### Confiança
Status de validação de uma entrada do catálogo:
- **draft** — criada ou auto-descoberta mas ainda não validada
- **validated** — confirmada precisa para uma versão específica do console
- **stale** — validada anteriormente mas o console mudou desde então
