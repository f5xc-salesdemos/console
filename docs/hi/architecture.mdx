## उद्देश्य

यह रिपॉजिटरी F5 Distributed Cloud (F5 XC) प्रशासन कंसोल UI का **स्कीमा-संचालित, मशीन-पठनीय इन्वेंटरी** है। यह API संसाधनों को उनके कंसोल स्थानों, नेविगेशन पथों, फ़ॉर्म संरचनाओं और चरण-दर-चरण ब्राउज़र स्वचालन वर्कफ़्लो में मैप करता है।

कैटलॉग का उपयोग xcsh के ब्राउज़र स्वचालन रनटाइम द्वारा किया जाता है:

- **बिक्री डेमो** — कंसोल CRUD ऑपरेशन के AI-सहायक वॉकथ्रू
- **प्रशिक्षण वीडियो** — YouTube और प्रस्तुतियों के लिए पुनरुत्पादनीय ब्राउज़र रिकॉर्डिंग
- **UAT/QA स्वचालन** — स्वचालित कंसोल इंटरैक्शन परीक्षण
- **AI-सहायक नेविगेशन** — AI सहायक यह पता लगा सकता है कि संसाधन कहाँ बनाना है

## एकीकरण बिंदु

| सिस्टम | एकीकरण |
|--------|-------------|
| api-specs-enriched | संसाधन `api.kind` मान मेल खाते हैं; स्ट्रिंग कुंजी द्वारा क्रॉस-संदर्भित |
| xcsh browser.ts | रनटाइम उपभोक्ता — YAML पढ़ता है, CDP के माध्यम से निष्पादित करता है |
| docs-control | गवर्नेंस — प्रबंधित फ़ाइलें, CI वर्कफ़्लो, ब्रांच सुरक्षा |
| docs-theme | कैटलॉग से दस्तावेज़ीकरण उत्पन्न |
