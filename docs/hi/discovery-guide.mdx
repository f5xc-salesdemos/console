xcsh ब्राउज़र स्वचालन उपकरण F5 XC कंसोल को क्रॉल करके कैटलॉग को कैसे भरता है।

## अवलोकन

कैटलॉग प्रविष्टियाँ **ब्राउज़र-फर्स्ट डिस्कवरी** दृष्टिकोण का उपयोग करके बनाई जाती हैं:

1. xcsh उपयोगकर्ता के Chrome ब्राउज़र से जुड़ता है (पोर्ट 9222 पर CDP के माध्यम से)
2. xcsh कंसोल को नेविगेट करता है और संरचनात्मक जानकारी निकालता है
3. xcsh कैटलॉग स्कीमा से मेल खाते YAML फ़ाइलें आउटपुट करता है
4. एक मानव प्रविष्टियों की समीक्षा करता है, परिष्कृत करता है और कमिट करता है

## डिस्कवरी के बाद सत्यापन

प्रत्येक डिस्कवरी रन के बाद:

1. स्कीमा अनुपालन जाँचने के लिए `npm run validate` चलाएँ
2. सटीकता के लिए उत्पन्न प्रविष्टियों की समीक्षा करें
3. विश्वास स्तर समायोजित करें
4. मानक PR वर्कफ़्लो के माध्यम से कमिट और पुश करें
