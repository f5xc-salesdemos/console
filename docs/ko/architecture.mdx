## 목적

이 저장소는 F5 Distributed Cloud(F5 XC) 관리 콘솔 UI의 **스키마 기반 머신 가독 인벤토리**입니다. API 리소스를 콘솔 위치, 탐색 경로, 폼 구조 및 단계별 브라우저 자동화 워크플로우에 매핑합니다.

카탈로그는 xcsh의 브라우저 자동화 런타임에 의해 소비되어 다음을 지원합니다:

- **영업 데모** — 콘솔 CRUD 작업의 AI 지원 안내
- **교육 동영상** — YouTube 및 발표용 재현 가능한 브라우저 녹화
- **UAT/QA 자동화** — 자동화된 콘솔 인터랙션 테스트
- **AI 지원 탐색** — AI 어시스턴트가 리소스를 생성할 위치를 찾아 브라우저를 안내

## 통합 지점

| 시스템 | 통합 |
|--------|-------------|
| api-specs-enriched | 리소스 `api.kind` 값이 일치; 문자열 키로 상호 참조 |
| xcsh browser.ts | 런타임 소비자 — YAML 읽기, CDP 통해 실행 |
| docs-control | 거버넌스 — 관리 파일, CI 워크플로우, 브랜치 보호 |
| docs-theme | 카탈로그에서 문서 생성 |
