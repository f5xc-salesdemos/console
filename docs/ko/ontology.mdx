콘솔 카탈로그에서 사용되는 용어 정의.

## 핵심 용어

### 라우트
특정 화면을 식별하는 콘솔 URL 경로 또는 패턴.

### 리소스
알려진 스키마, CRUD 엔드포인트 및 콘솔의 하나 이상의 위치를 가진 API 정의 객체 유형.

### 워크플로우
리소스에 대해 정확히 하나의 작업을 수행하는 순서가 지정된 자동화 단계의 시퀀스.

### 신뢰도
카탈로그 항목의 검증 상태:
- **draft** — 작성되거나 자동 발견되었지만 아직 검증되지 않음
- **validated** — 특정 콘솔 버전에 대해 정확함이 확인됨
- **stale** — 이전에 검증되었지만 그 이후 콘솔이 변경됨
