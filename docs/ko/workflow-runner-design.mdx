이 카탈로그에서 워크플로우 YAML을 읽고 Chrome DevTools Protocol을 통해 단계를 실행하는 xcsh 컴포넌트 설계.

## 액션 매핑

| 워크플로우 액션 | Browser Tool 액션 | 참고 |
|---|---|---|
| `navigate` | `goto` + `wait_for_selector` | baseURL을 상대 URL 앞에 추가 |
| `click` | `click` | 직접 선택자 매핑 |
| `fill` | `fill` | 값에서 {param} 해결 |
| `assert` | `get_text` + 문자열 비교 | 불일치 시 즉시 실패 |

## 오류 처리

- 즉시 실패: 첫 번째 실패 단계에서 실행 중지
- 실패 스크린샷 (최선 노력)
- 재시도 로직 없음
