xcsh 브라우저 자동화 도구가 F5 XC 콘솔을 크롤링하여 카탈로그를 채우는 방법.

## 개요

카탈로그 항목은 **브라우저 우선 디스커버리** 방식을 사용하여 작성됩니다:

1. xcsh가 사용자의 Chrome 브라우저에 연결합니다(포트 9222의 CDP 통해)
2. xcsh가 콘솔을 탐색하여 구조 정보를 추출합니다
3. xcsh가 카탈로그 스키마와 일치하는 YAML 파일을 출력합니다
4. 사람이 항목을 검토, 개선, 커밋합니다

## 디스커버리 후 유효성 검사

1. `npm run validate` 실행으로 스키마 준수 확인
2. 생성된 항목의 정확성 검토
3. 신뢰도 수준 조정
4. 표준 PR 워크플로우를 통해 커밋 및 푸시
