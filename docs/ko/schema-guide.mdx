카탈로그 항목 작성 및 유지 관리 방법.

## 신뢰도 수준

| 수준 | 의미 | 사용 시기 |
|-------|---------|-------------|
| `draft` | 미검증 | 초기 작성, 자동 발견 항목 |
| `validated` | 확인됨 | 지정된 버전의 라이브 콘솔에 대해 검증됨 |
| `stale` | 오래됨 | 마지막 검증 이후 콘솔이 변경됨 |

## 유효성 검사

`npm run validate`를 실행하여 모든 카탈로그 항목을 스키마에 대해 확인합니다.
