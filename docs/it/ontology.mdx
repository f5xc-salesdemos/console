Definizioni dei termini usati nel catalogo della console.

## Termini principali

### Route
Un percorso URL della console o pattern che identifica uno schermo distinto.

### Risorsa
Un tipo di oggetto definito da API con schema noto, endpoint CRUD e una o più posizioni nella console.

### Workflow
Una sequenza ordinata di passaggi di automazione che esegue esattamente un'operazione su una risorsa.

### Confidenza
Stato di validazione di una voce del catalogo:
- **draft** — creata o auto-scoperta ma non ancora validata
- **validated** — confermata accurata per una versione specifica della console
- **stale** — precedentemente validata ma la console è cambiata da allora
