Come creare e mantenere le voci del catalogo.

## Livelli di confidenza

| Livello | Significato | Quando usare |
|-------|---------|-------------|
| `draft` | Non verificato | Creazione iniziale, voci auto-scoperte |
| `validated` | Confermato | Verificato contro la console live alla versione indicata |
| `stale` | Obsoleto | La console è cambiata dall'ultima validazione |

## Validazione

Eseguire `npm run validate` per verificare tutte le voci del catalogo rispetto ai loro schemi.
